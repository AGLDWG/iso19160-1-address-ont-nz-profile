Address Componentc

IRI http://linked.data.gov.au/def/iso19160-1-address-nz-profile#AddressComponent
Description ISO 19160 Address Component class with the additional restrictions of requiring one and only one Address lifespan, Address provenance, Address Component lifecycle stage and an Address Component Value locale properties
Super-classes iso19160-1:AddressComponentc
Restrictions iso19160-1:Address.lifespan exactly 1
iso19160-1:Address.provenance exactly 1
iso19160-1:AddressComponentValue.locale exactly 0
iso19160-1:AddressComponent.lifecycleStage exactly 1

Addressable Objectc

IRI http://linked.data.gov.au/def/iso19160-1-address-nz-profile#AddressableObject
Description ISO 19160 Addressable Object class with the additional restrictions of requiring one and only one Address Component lifespan, Address Component position and an Address Component lifecycle stage properties.
Super-classes iso19160-1:AddressableObjectc
Restrictions iso19160-1:Address.lifespan exactly 1
iso19160-1:AddressComponent.lifecycleStage exactly 1
iso19160-1:Address.position exactly 0

Address Periodc
